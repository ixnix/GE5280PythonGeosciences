# GE5280: Python Programming for Geosciences

## Course Description

This course introduces Python as a versatile programming language for geoscience applications. Designed for graduate students with no prior programming experience, it covers core programming concepts and progresses to advanced techniques for handling and analyzing structured geoscience datasets. Topics include variables, data structures, control flow, functions, NumPy, Pandas, time series analysis, object-oriented programming, and scientific visualization with Cartopy and Seaborn.

## Getting Started

1. Install [Anaconda](https://www.anaconda.com/download) or [Miniconda](https://docs.conda.io/en/latest/miniconda.html).
2. Launch JupyterLab:
   ```bash
   jupyter lab
   ```
3. Navigate to a module folder and open the lecture notebook.

Each module folder contains:
- **Lecture notebook** — instructional material with worked examples
- **Assignment notebook(s)** — practice problems for students
- **`data/`** — datasets used in the module (where applicable)
- **`img/`** — images and diagrams (where applicable)
